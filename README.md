# Practical1

This repository contains the first FSD practical with four HTML/CSS exercises:

- `Q1_Text_Formatting`
- `Q2_Student_Profile`
- `Q3_Restaurant_Menu`
- `Q4_HTML_Forms`

## Open Locally

Open any `index.html` file directly in a browser, or run a simple local server from the `Practical1` folder:

```powershell
npx serve .
```
