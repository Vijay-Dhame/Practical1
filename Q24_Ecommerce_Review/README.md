# Q24_Ecommerce_Review

This is a full MERN stack assignment (React + Node + MongoDB).

## Quick Run

### Step 1: Start the Backend
Open a terminal in the project root and run:
```powershell
cd FSD_Assignments/Assignment_7/Q24_Ecommerce_Review; node server.js
```

### Step 2: Start the Frontend
Open a **new terminal** in the project root and run:
```powershell
cd FSD_Assignments/Assignment_7/Q24_Ecommerce_Review/frontend; npm start
```
